    Authorizes client with given access token required in restricted endpoints.

    Postcondition

    Postcondition:

    • If a client is authorized successfully NSNotification.Name.AtlasAuthorized is posted on NotificationCenter.default, otherwise it is NSNotification.Name.AtlasDeauthorized.
    • NSNotification.Name.AtlasAuthorizationChanged is always posted regadless the result.
